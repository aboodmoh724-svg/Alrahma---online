# Al-Rahma Reports - Project Rules

## 1. Syria vs Remote/Online Section Separation

### سوريا (ONSITE_SYRIA)
- الصفحات تحت: `app/syria/`
- قالب الرسالة: `syriaDailyReportMessage()` داخل API مخصص (لا يستخدم `renderMessageTemplate`)
- الشعار: `إدارة تحفيظ الرحمة للقرآن الكريم - سوريا`
- حقل "آخر 5 صفحات" (`lastFiveMemorized`) **غير مستخدم** في سوريا وينبغي إخفاؤه دائماً
- حقل "عدد الصفحات" **غير مستخدم** في واجهة معلمي سوريا — يعتمدون على أرقام الآيات فقط
- أرقام الهواتف تُعالَج بكود الدولة `963` (سوريا) افتراضياً
- الأرقام المحلية ذات الـ 10 أرقام (تبدأ بـ 0) تحتاج normalization قبل الفحص

### الأونلاين (REMOTE)
- الصفحات تحت: `app/remote/`
- قالب الرسالة: `renderMessageTemplate("REMOTE_REPORT", ...)` من قاعدة البيانات
- الشعار: `إدارة منصة الرحمة لتعليم القرآن الكريم`
- جميع الحقول (آخر 5 صفحات، عدد الصفحات...) مفعّلة

### القاعدة
- **لا تعدّل أي إعدادات أو قوالب في صفحة `/remote/admin/messages` عند العمل على سوريا**
- **لا تعدّل أي إعدادات أو قوالب خاصة بالأونلاين عند العمل على سوريا، والعكس صحيح**
- صفحة قوالب سوريا المخصصة هي: `/syria/admin/messages`

## 2. Note Presets - Syria Filter
- مصدر الملاحظات الجاهزة لسوريا هو نفس API الأونلاين: `/api/report-note-presets`
- عند عرضها في واجهة سوريا، يجب **تصفية** أي ملاحظة تحتوي على "خمس" أو "5" لأن خاصية آخر 5 صفحات غير موجودة في سوريا
- الفلتر: `.filter((item) => !item.includes("خمس") && !item.includes("5"))`

## 3. Test Student (studentCode: "7500")
- الطالب التجريبي له كود `7500` ورقم هاتفه هو رقم صاحب المشروع الشخصي
- يجب أن يتمكن أي معلم مسجل دخوله من الوصول إليه (للاختبار)
- في جميع استعلامات `prisma.student.findFirst` المرتبطة بالتقارير، يجب إضافة:
  ```ts
  OR: [
    { teacherId },
    { studentCode: "7500" },
  ]
  ```
- هذا يُطبَّق في: `reports/route.ts`، `students/[id]/history/route.ts`، `students/[id]/progress/route.ts`
- الطالب التجريبي مستثنى من قيود التعديل والتكرار

## 4. Student Progress Section - Optional
- قسم "بداية الطالب مع المعلم" في صفحة التقرير اليومي لسوريا هو **اختياري بالكامل**
- عدم حفظه لا يمنع حفظ التقرير اليومي

## 5. WhatsApp Report Message Format - Syria
صيغة الرسالة النهائية المعتمدة:
- العنوان: `تقرير الطالب اليومي` (بدون "قسم سوريا")
- الواجب يُقسَّم إلى نقاط منفصلة عبر `split(" | ")`
- **لا** يُذكر "عدد الصفحات" في الواجب
- التقييم يشمل الدرس الجديد والمراجعة فقط (لا يشمل آخر 5 صفحات)
- التوقيع: `إدارة تحفيظ الرحمة للقرآن الكريم - سوريا`
